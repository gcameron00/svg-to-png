# SVG to PNG

A simple, client-side web tool for converting SVG files to PNG images. No uploads, no server — everything happens in your browser.

## What it does

Drop an SVG file onto the page (or use the file picker), and instantly download it as a PNG. That's it.

## Tech

- Static site hosted on Cloudflare Pages
- Pure HTML/CSS/JS — no dependencies, no build step

## Development

Open `index.html` in a browser, or deploy to any static host.
